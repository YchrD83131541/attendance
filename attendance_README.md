# attendance.py README

## 概要

`attendance.py` は、スポーツクラブ向けの勤怠管理 GUI アプリケーションです。  
Python の Tkinter を使ったデスクトップ画面から、出退勤の記録・修正・削除、レッスン数管理、36協定チェック、有給休暇管理、Excel 出力などを行えます。

---

## 主な機能

| 機能 | 説明 |
|------|------|
| ユーザー認証 | ログイン画面でユーザー名・パスワードを確認。SHA-256 ハッシュで保存 |
| スタッフ管理 | スタッフの追加・削除。ドロップダウンで選択式入力 |
| 出退勤登録 | 出勤・退勤時刻、休憩時間、事務作業時間を記録 |
| レッスン管理 | メイン/サブ担当別のレッスン数を登録 |
| 勤務区分 | 通常出勤・所定休日出勤・法定休日出勤を区別して記録 |
| 訂正・削除 | 既存レコードの修正（日付変更含む）・削除 |
| 月別集計 | スタッフ別の勤務時間・レッスン数をポップアップで確認 |
| Excel 出力 | 月選択後に「勤怠詳細」「スタッフ別集計」2 シートの xlsx を生成 |
| 36協定管理 | 月間・年間（720h）・複数月平均の時間外労働チェック、CSV 出力 |
| 有給休暇管理 | 付与・取得の登録と残日数管理。年 5 日取得義務のアラート表示 |
| ユーザー管理 | アカウントの作成・削除・パスワード変更 |

---

## 依存ライブラリ

### Python 標準ライブラリ（追加インストール不要）
- `datetime`, `json`, `os`, `tkinter`, `hashlib`, `hmac`

### 外部ライブラリ
- `openpyxl`（Excel 出力に使用）

### プロジェクト内モジュール
- `labor.py`（勤怠データの読み書き・計算ロジック・36協定管理）

### インストール
```powershell
pip install openpyxl
```

## 起動方法

```powershell
python attendance.py
```

初回起動時、管理者アカウント `admin` / `admin` が自動作成されます。  
**セキュリティのため、初回ログイン後にパスワードを変更してください。**

| ファイル | パス | 内容 |
|----------|------|------|
| ユーザー情報 | `%USERPROFILE%\Documents\AttendanceApp\users.json` | ログインアカウント（ハッシュ化済み） |

## 画面操作ガイド

### 1. ログイン
1. アプリ起動後、ログイン画面が表示されます
2. ユーザー名とパスワードを入力して「ログイン」をクリック
3. 認証に成功するとメイン画面が開きます

### 2. メイン画面の入力項目

| 項目 | 説明 |
|------|------|
| スタッフ名 | ドロップダウンから選択 |
| 担当 | メイン / サブ を選択 |
| 勤務区分 | 通常出勤 / 所定休日出勤 / 法定休日出勤 |
| 日付 | `YYYY-MM-DD` 形式で入力（デフォルト: 今日） |
| レッスン数 | 担当するレッスン数（整数） |
| 休憩（分） | 休憩時間（分単位） |
| 事務作業 | 事務作業時間（時間・分で入力） |
| 出勤時間 | `YYYY-MM-DD HH:MM:SS` または `HH:MM` 形式 |
| 退勤時間 | 同上 |

### 3. ボタン一覧

| ボタン | 機能 |
|--------|------|
| 登録 | 入力内容をまとめて保存（既存レコードは上書き確認） |
| レッスン追加 | 選択スタッフの指定日にレッスン数を追加 |
| 訂正保存 | 選択行のレコードを修正して保存 |
| 削除 | 選択行のレコードを削除 |
| 更新 | 一覧を最新データで再描画 |
| 集計 | スタッフ別の累計勤務時間・レッスン数をポップアップ表示 |
| 36協定管理 | 月間・年間の時間外労働チェック画面を開く |
| ユーザー管理 | ログインアカウントの追加・削除・パスワード変更 |
| Excel出力 | 月を選択して xlsx ファイルを生成・保存 |
| 有給管理 | 有給休暇の付与・取得・残高管理画面を開く |
| 名前順ソート | 一覧表示を名前順 ↔ 日付順で切り替え |
| スタッフ管理 | スタッフの追加・削除 |

## 36協定管理

メイン画面の「36協定管理」ボタンから開く専用ウィンドウです。

| 操作 | 説明 |
|------|------|
| 表示 | 指定年月のスタッフ別時間外労働を一覧表示 |
| CSV出力 | 月次サマリーを CSV ファイルに書き出し |
| 年間CSV出力 | 当年の全月分を CSV で書き出し |
| 複数月CSV出力 | 指定末月から N ヶ月分を CSV で書き出し |
| 年間720チェック | 年間 720 時間超過者をダイアログで通知 |
| 複数月平均チェック | 複数月平均の時間外が 80h を超えるスタッフを一覧表示 |

起動時に当月の時間外が 45 時間以上のスタッフがいる場合、自動で警告ダイアログが表示されます。

---

## 有給休暇管理

「有給管理」ボタンから開くウィンドウです。4 つのタブで構成されています。

| タブ | 機能 |
|------|------|
| 残高一覧 | 基準日時点の付与日数・取得日数・残日数を一覧表示。年 5 日義務の達成状況も確認可能 |
| 付与登録 | スタッフへの有給付与日数・有効期限を登録 |
| 取得登録 | 有給取得日・日数（0.5 日単位）を登録。残日数超過時は確認ダイアログを表示 |
| 履歴一覧 | スタッフ・年でフィルタリングして付与・取得履歴を一覧表示 |

---

## Excel 出力フォーマット

「Excel出力」ボタンで、対象年月を選択して `勤怠集計_YYYY-MM.xlsx` を生成します。

**シート1: 勤怠詳細**  
日付・スタッフ名・勤務区分・出退勤時刻・休憩・総勤務時間・メイン/サブレッスン数・事務作業時間

**シート2: スタッフ別集計**  
通常出勤・所定休日出勤・法定休日出勤の日数と時間を区分別に集計。  
メイン/サブ/総レッスン数および事務作業時間も含む合計行付き。

---

## 注意点

- `labor.py` が同じディレクトリに必要です。
- `openpyxl` がインストールされていない場合、Excel 出力機能が使えません。
- 管理者アカウント `admin` は削除できません。
- 既存レコードを「登録」で上書きする場合、確認ダイアログが表示されます。
- 有給の残日数が不足している場合も、確認後に登録できます。
